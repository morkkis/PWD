Typescript

TypeScript is a typed superset of JavaScript that compiles to plain JavaScript (It’s a bit like what SCSS is for CSS.)

  • Created by Microsoft
  • Open source language, developed in public at TypeScript Git

Why Typescript?

  • TypesProvide an optional type system for JavaScript
  • Future FeaturesProvide planned features from future JavaScript editions to current JavaScript engines
  • IDE auto completion
  • Object-oriented
  • AOT compilation
  • Types are documentation
  • Not locked in
  • Ability to choose ECMAScript version
